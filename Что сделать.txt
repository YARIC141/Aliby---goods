Нужно сделать чеклист абонементов на экране абонементов в разделе активные:
Чеклист содержит список категорий заведений
- Чеклист содержит список категорий товаров и услуг
  - Если по заданной категории есть абонемент то эта строка зеленая
  - Если по заданной категории нет абонемента то эта строка серая
При выпуске каждого абонемента заставлять админа подписывать договор о выполнении объема обязательств абонемента
Для еды если заказ происходит вне времени работы по соответствующему часовому поясу то запрещать заказ.
